# Smart Company

HCMUT CO3109 project Smart Company.

## Instruction

1. Create a `.env.local` file in the root directory of the project.

```env
API_URL=
```

2. Install dependencies

```sh
npm install
```

3. Run the project

```sh
npm run dev
```

## Workflows

1. Generate OpenAPI client

```sh
npm run openapi
```
